# 한 수 — AI 사주 라이프 코치

## 1. 도메인 선택 이유

한국 사주/운세 시장 1.4조원. MZ세대가 전문 플랫폼 사용자의 80%. AI 사주 이용률(40.7%)이 오프라인 점술(18.5%)을 이미 압도. 그러나 기존 서비스(점신, 포스텔러, 플롯)는 모두 **"해석 리포트 자판기"**에 머물러 있다. 서양에서는 Co-Star(DAU 30-40%), Replika(3,500만 유저)가 **AI 컴패니언 모델**로 거대한 성공을 거뒀지만, 한국 사주 도메인에는 아직 적용되지 않았다. 사주라는 자기 이해의 프레임워크를 AI 라이프 코치의 렌즈로 재정의한다.

## 2. 타겟과 문제

**타겟**: 2030 직장인. 사주앱을 2-3개 병행하며 "다 비슷하다"고 느끼는 사람. 사주를 보고도 "그래서 나는 뭘 해야 하지?"라는 질문이 남는 사람.

**문제 3가지**:
1. **정확도-습관 패러독스** — 점신 MAU 95만이지만 "안 맞아도 습관처럼 본다." 더 나은 대안이 없어서 머무른다.
2. **해석-행동 단절** — "올해 관재수 조심" 같은 해석은 있지만, 구체적으로 뭘 하라는 건지 알 수 없다. 사용자가 원하는 건 ①감정 확인 ②행동 허가 ③구체적 방향인데, 기존 앱은 ①만 겨우 해준다.
3. **맥락 없는 AI** — ChatGPT 사주는 매번 생년월일 재입력, 사주 계산 부정확, 사용자 상황과 결합 없음.

## 3. 핵심 기능 4가지

**F1. 내 사주 프로필** — KASI 기반 만세력 라이브러리로 정확한 사주팔자 계산. 오행 분포, 일간 성격, 용신, 오늘의 일진을 한눈에. → "이 앱은 나를 안다"는 첫 신뢰.

**F2. 코칭 카드 상담** — 자유 텍스트로 고민 입력 → 사주 원국 + 오늘 일진 + RAG 명리학 지식 결합 → 코칭 카드(진단/한 수/타이밍/피할 것/명리학 근거) 반환. Claude tool_use로 5필드 구조를 강제하여 UX 품질 일정. → "읽고 끝"이 아닌 "실행할 수 있는" 조언.

**F3. 캐릭터 3종** — 같은 사주 데이터를 다른 해석 프레임으로 읽는 3개 캐릭터: 친한 언니/형(감정 프레임), 할머니 역술가(오행 프레임), 팩트체커(십신 프레임). 톤만 다른 게 아니라 해석 렌즈 자체가 다르다. → 같은 고민을 다른 관점으로 재질문하게 만들어 세션 깊이 증가.

**F4. 한 수 일지 + 피드백** — 과거 상담 타임라인 + "그 한 수, 해봤어요?" 피드백(좋았다/별로/아직). → 피드백 루프가 일회성 소비를 관계 형성으로 전환.

## 4. 성공 지표

| 지표 | 목표 | 실패 시 액션 |
|---|---|---|
| **세션당 상담 수** (북극성) | ≥ 1.5 | 코칭 카드 품질 이터레이션 |
| 캐릭터 전환 경험률 | ≥ 30% | 캐릭터 간 프레임 차이 강화 |
| 프로필→상담 전환율 | ≥ 60% | 오늘의 일진 / 프로필 UX 개선 |
| 피드백 응답률 | ≥ 20% | 피드백 타이밍/UX 변경 |
