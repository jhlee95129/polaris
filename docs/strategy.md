# 한수 — 서비스 전략서

## 1. 서비스를 만들게 된 동기

### 핵심 관찰
사주/운세 시장은 크지만(1.4조원), **"해석을 파는 시대"에 머물러 있다.** 사용자가 실제로 원하는 건 해석이 아니라 **"그래서 나는 지금 뭘 해야 하지?"에 대한 답**이다.

서양에서는 Co-Star(DAU 30-40%), Replika(3,500만 유저), Character.AI가 **AI 컴패니언 모델**로 거대한 성공을 거뒀다. Replika 사용자의 85%가 AI와 감정적 유대를 형성하며, 감정적 지지(44.6%)와 동반자 관계(77.1%)가 핵심 가치.

**한국에는 이 모델이 사주 도메인에 아직 적용되지 않았다.**

### 리프레이밍: "사주 풀이" → "AI 사주 라이프 코치"
사주는 본질적으로 **자기 이해의 프레임워크**다. "올해 관재수 조심" 같은 예측이 아니라, "나는 왜 이런 상황에서 이렇게 반응하는가"를 설명하는 렌즈다. 이것은 심리상담, 라이프 코칭과 본질적으로 같다.

> 기존 사주 앱은 "신경정신과 진단서"를 판다. 한수는 "나를 가장 잘 아는 상담사와의 대화"를 판다.

## 2. 시장 분석

### 시장 규모
- **한국 사주/운세 시장**: 약 1.4조원 (2024년 추산)
- **글로벌 점성술 앱 시장**: 2024년 40.2억 달러 → 2033년 298.2억 달러 (CAGR 24.9%)
- **한국인 70%**가 매년 운세를 확인
- **MZ세대**가 전문 플랫폼 사용자의 **80%**
- 10~39세 평균 연간 점술 지출: **8만원**

### AI 사주의 부상
- AI 사주 이용률 **40.7%** vs 오프라인 점술 방문 **18.5%** — AI가 이미 이기고 있음
- ChatGPT 사주 풀이가 바이럴 → AI 사주에 대한 수용도가 높아진 상태
- 그러나 ChatGPT의 한계(매번 재입력, 부정확한 계산, 구조 없는 답변)가 명확

### 시장 점유율
| 서비스 | MAU/매출 | 점유율 | 비고 |
|---|---|---|---|
| 점신 | MAU 95만, 매출 830억 | 46.5% | 시장 1위, 관성적 사용 |
| 포스텔러 | 매출 100억+ | ~8% | 캐릭터 IP (운세남녀) |
| 헬로우봇 | MAU 30만+ | ~3% | AI 채팅형 |
| 뤼튼 플롯 | - | - | 유료 리포트 (열쇠 결제) |
| 기타 | - | - | 신점, 삼재력, 천기만세력 등 |

## 3. 경쟁사 분석

### 점신 (시장 1위)
- **형태**: 일방향 리포트 (신년운, 오늘의 운세, 궁합 등)
- **강점**: 브랜드 인지도, MAU 95만
- **약점**: "솔직히 하나도 안 맞는다, 근데 습관처럼 봄" — 정확도 불만이 높지만 대안 부재
- **기회**: 브랜드 관성으로 1위를 유지하고 있어 디스럽션 가능

### 포스텔러
- **형태**: 캐릭터(운세남녀) 기반 리포트
- **강점**: IP 캐릭터로 친근감
- **약점**: "사주가 제일 안 맞는다" 리뷰 다수. 캐릭터는 있지만 대화/관계가 없음
- **기회**: 캐릭터가 있지만 "관계"를 형성하지 못함 → 우리의 차별점

### 뤼튼 플롯 (flot.wrtn.ai)
- **형태**: 카테고리별 유료 리포트 판매 (열쇠 결제)
- **강점**: 뤼튼 AI 기술력, 깔끔한 UI
- **약점**: 일방향 소비(읽고 끝), 사용자 상황 맥락 없음, 결제 장벽
- **한수와의 차별점**:

| 플롯 | 한수 |
|---|---|
| 카테고리 선택 → 리포트 열람 | 자유 텍스트로 고민 입력 |
| 누구에게나 같은 리포트 | 같은 사주라도 상황에 따라 다른 답 |
| 읽고 끝 (일방향) | 캐릭터 전환 + 피드백 루프 |
| 결제 장벽 | 무료 (MVP) |

### Co-Star / Replika (글로벌 레퍼런스)
- Co-Star: 일일 점성술 알림 → DAU 30-40%. "매일 대화하는 습관" 형성에 성공
- Replika: 3,500만 유저, 85%가 AI와 감정적 유대. "AI 친구"로 월 $20 과금
- **핵심 인사이트**: 콘텐츠가 아닌 **관계**가 리텐션과 매출을 만든다

## 4. 차별화 전략 — 3축

### 축 1: AI 컴패니언 모델
Co-Star/Replika의 "감정적 유대 + 일상적 터치포인트" 전략을 한국 사주에 적용. 캐릭터가 단순 페르소나가 아니라 **나와 관계를 쌓아가는 코치**.

### 축 2: RAG 기반 명리학 전문 지식
적천수, 궁통보감 등 명리학 고전을 벡터DB에 임베딩. "AI가 지어낸 해석"이 아닌 **"명리학 근거에 기반한 코칭"** 제공. 기술적으로도 Product Engineer의 역량을 증명.

### 축 3: 상황 맥락 + 사주 원국 + 전문 지식 3중 결합
기존 앱: 사주 데이터만으로 해석
ChatGPT: 사용자 상황만으로 조언
**한수: 사주 + 상황 + 명리학 지식을 Claude tool_use로 구조화하여 결합**

## 5. 습관 형성 → 매출 연결 전략

### 습관 형성 루프
```
오늘의 일진 한 줄 → 앱 오픈 (Daily Hook)
  ↓
코치에게 오늘의 고민 → 대화 시작 (Engagement)
  ↓
코칭 카드 + "더 자세히" → 깊이 있는 경험 (Value)
  ↓
"해봤어요?" 피드백 → 재방문 이유 (Retention)
  ↓
반복 → 관계 형성 → 의존 → 유료 전환
```

### 매출 시나리오 (MVP 이후)
- **무료**: 하루 3회 코칭 카드 + 오늘의 일진
- **프리미엄**: 무제한 코칭 + 캐릭터 전체 해금 + 대운/세운 심화 분석

### 사업 성공률 논거
1. **시장 검증 완료**: 점신(MAU 95만), 포스텔러(매출 100억+)가 시장 존재 증명
2. **모델 검증 완료**: Co-Star, Replika가 AI 컴패니언 모델의 수익성 증명
3. **갭 존재**: 한국 사주 × AI 컴패니언 조합은 미개척 영역
4. **기술 우위**: RAG + 만세력 + tool_use 구조화 — 단순 ChatGPT 래퍼와 차별화

## 6. 핵심 성공 지표

| 지표 | 목표 | 가설 |
|---|---|---|
| **세션당 상담 수** (북극성) | ≥ 1.5 | 실행 가능한 코칭이면 더 물어본다 |
| 캐릭터 전환 경험률 | ≥ 30% | 다른 관점이 재질문을 유발한다 |
| 프로필→상담 전환율 | ≥ 60% | 정확한 프로필이 신뢰를 만든다 |
| 피드백 응답률 | ≥ 20% | 피드백이 재방문 이유가 된다 |
| D7 재방문율 | ≥ 15% | 기존 앱 평균 <10% 상회 |
