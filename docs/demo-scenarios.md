# 한 수 — 데모 시나리오

> 평가자가 5분 안에 핵심 경험을 할 수 있는 가이드입니다.

## 사전 준비

1. `pnpm install && pnpm dev` 로 로컬 서버 실행
2. `.env.local`에 `ANTHROPIC_API_KEY` 설정 (필수)
3. `http://localhost:3000` 접속

---

## 시나리오 1: 이직 고민 (2030 직장인)

### Step 1. 생년월일 입력
- **양력 1995년 3월 22일, 오시 (11:30~13:30)**
- 성별: 여성

### Step 2. 사주 프로필 확인
- 사주팔자 4주 확인 (한자 + 한글)
- 오행 분포 차트에서 어떤 오행이 강하고 약한지 확인
- "AI 프로필 분석" 버튼 클릭 → Claude가 성격/강점/주의점/용신 조언/오늘 한 줄 생성

### Step 3. 코칭 상담 — "친한 언니/형" 캐릭터
- "고민 상담받기" 클릭 → 상담 페이지
- 캐릭터: **친한 언니/형** 선택
- 질문: **"2년차인데 이직을 고민하고 있어요. 연봉은 오르는데 성장하는 느낌이 없어요."**
- 코칭 카드 확인: 진단(사주 근거 포함) → 한 수(구체적 행동) → 타이밍 → 피할 것 → 명리학 근거

### Step 4. 캐릭터 전환 — 같은 질문, 다른 관점
- "다른 관점으로 보기" 클릭 → 다른 캐릭터로 자동 전환
- **같은 고민이지만 해석 프레임이 달라지는 것**을 확인
  - 언니/형: 감정 공감 + 관계 프레임
  - 할머니: 오행 직설 + 생활 조언
  - 팩트체커: 십신/용신 논리 분석

## 시나리오 2: 연애 상담 (MZ 싱글)

### Step 1. (이미 프로필이 있으면 그대로 사용)

### Step 2. 코칭 상담 — "동네 할머니 역술가" 캐릭터
- 캐릭터: **동네 할머니 역술가** 선택
- 질문: **"내일 소개팅인데, 어떤 준비를 하면 좋을까요?"**
- 할머니 스타일의 단호한 조언 + 오행 기반 구체적 생활 팁(색상, 시간대 등)

### Step 3. 코칭 상담 — "팩트체커" 캐릭터로 재질문
- 캐릭터: **팩트체커** 선택
- 같은 질문 입력
- 분석적 관점의 조언 + 십신/용신 관계 근거

---

## 시나리오 3: 일지 + 피드백

### Step 1. 상담 일지 확인
- 하단 네비게이션 → "일지" 탭
- 지금까지의 상담 기록이 타임라인 형태로 표시

### Step 2. 피드백 제공
- 상담 기록 클릭하여 펼치기
- "그 한 수, 해봤어요?" 질문에 "좋았어요" / "별로였어요" / "아직이요" 응답
- 피드백이 기록되고, 통계에 반영됨

---

## 핵심 체크포인트

평가 시 확인해주시면 좋은 포인트들:

1. **사주 정확도**: 오행 분포, 일간, 띠가 만세력닷컴 등과 일치하는지
2. **코칭 카드 품질**: "조심하세요" 같은 모호한 표현 대신 구체적 행동이 나오는지
3. **캐릭터 차이**: 같은 질문에 대해 3개 캐릭터가 실제로 다른 관점을 제시하는지
4. **구조 일관성**: 모든 코칭 카드가 5필드(진단/행동/타이밍/피할 것/근거) 구조를 유지하는지
5. **UX 흐름**: 온보딩 → 프로필 → 상담 → 일지의 자연스러운 플로우
